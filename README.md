# Atlan Python Client

Python library for Atlan.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install Python client for Atlan

```bash
> pip install pyatlan
```

Verify if atlan client is installed:
```bash
> pip list

Package      Version
------------ ---------
pyatlan      0.0.1
```
